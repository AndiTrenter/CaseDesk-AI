## Wichtige Hinweise zur Konfiguration

### KI-Konfiguration (OpenAI)
Um die KI-Funktionen (automatische Termin-/Fristenextraktion) zu aktivieren:
1. Einloggen als Admin
2. Gehe zu Einstellungen → KI-Konfiguration
3. Wähle "OpenAI (ChatGPT)" als Provider
4. Gib deinen OpenAI API-Key ein
5. Klicke "Speichern"

### Automatische Termin-/Fristenextraktion
Wenn die KI konfiguriert ist:
- Beim Dokumenten-Upload werden Fristen automatisch erkannt
- Aufgaben und Kalendereinträge werden automatisch erstellt
- Der KI-Assistent kann Termine/Aufgaben aus dem Chat erstellen
